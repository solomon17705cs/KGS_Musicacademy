# Changelog

All notable changes to KGS Music Academy will be documented here.

## [1.1.0] — 2026-06-13

### Added
- Batch capacity thresholds with red indicator on class-days pages
- Parent push notifications when teacher reviews student progress
- In-app notification for parent when progress is saved

### Changed
- Removed Sunday from day selector buttons
- Only individual over-capacity instrument counts turn red (not the whole string)
- Instrument name casing normalized in class-days view

### Removed
- Attendance field from edit-progress pages

### Added
- Changes in leaderboard page

### Changed
